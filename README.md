# Resizable

Easy resizable layout html with javascrip and jquery

##Resize vertically
```js
var bar = $('.drag');

var topElement = $('.top');

var botElemnt = $('.bottom');

resizeElementsWithResizeBar('.drag', '.top', '.bottom');

```

##Resize horizontally
```js
var bar = $('.drag');

var leftElement = $('.top');

var rightElemnt = $('.bottom');

resizeElementsWithResizeBar('.drag', '.top', '.bottom', 'horizontal');

```
